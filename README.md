# Sudoku-Master
Sudoku Solver ; Game built with Python and wxPython using Backtracking, MRV heuristic, and bitmask optimization.
# Features
# Solver mode
- Solve any valid Sudoku puzzle
- Optimized Backtracking algorithm with:
   - MRV (Minimum Remaining Values) ,
   - heuristic,
   - Bitmask-based constraint checking
- Optional animated solving
- Adjustable solving speed
- Real-time rule validation with error feedback
# Player Mode

 Solve any valid Sudoku puzzle
Optimized Backtracking algorithm with:
MRV (Minimum Remaining Values) ,
heuristic,
Bitmask-based constraint checking
Optional animated solving
Adjustable solving speed
Real-time rule validation with error feedback

# GUI

Clean and modern UI using wxPython
Keyboard navigation (arrow keys, enter, backspace)
Visual highlights for valid and invalid moves
Responsive window resizing
# How to run

Install dependencies: pip install wxPython
Run the program: python sudoku_solver.py in the terminal
